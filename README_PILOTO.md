# GNVIC piloto taller - NARZU

Esta carpeta es una copia reducida de GNVIC para el piloto del taller.

- Carga solo el artefacto NARZU.
- Usa una clave propia de `localStorage`: `gnvic_piloto_taller_data_v1`.
- Cada episodio se fija en 6 viñetas.
- Cada viñeta permite hasta 3 iteraciones de imagen.
- La creación y el borrado de artefactos están bloqueados para evitar desviar el flujo del taller.
- No requiere compilación, frameworks ni dependencias nuevas.

Para probarla, abre `index.html` desde esta carpeta o publícala como sitio estático.
